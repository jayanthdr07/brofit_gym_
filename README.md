# 🏋️ BROFIT GYM – User Interactive Fitness Website
.
BROFIT GYM is a gym-based interactive web application developed using **HTML, CSS, PHP, and MySQL**.  
The project focuses on user interaction, fitness profile management, and basic gym-related content such as training, nutrition, and equipment.


---
## 🚀 Features


- User Registration & Login system
- Personal Fitness Profile creation
- Gym equipment information
- Training & nutrition guidance pages
- Membership-related pages
- Motivational and inspiration content
- Clean and responsive UI
- PHP backend with MySQL database integration

## 🛠 Technologies Used

- **Frontend:** HTML5, CSS3  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP / WAMP / Localhost  

---
## 🔐 Database

- MySQL is used to store:

- User registration details

- Fitness profile information

- Database connection is handled using PHP (mysqli) with prepared statements.
